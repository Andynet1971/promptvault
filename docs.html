Use your own API key

PromptVault works out of the box: every account gets 3 free AI improvements, and Pro plans include a generous monthly quota. If you want unlimited usage, full control over your data, or to skip plans entirely, plug in your own provider key — it takes about 60 seconds.

Why bring your own key?

  • Unlimited improvements — no monthly cap, you only pay (or don't) the provider directly.
  • Full data control — requests go straight from your browser to the provider you choose.
  • Pick your model — fast and free with Groq or Gemini, top-tier quality with GPT-4o or Claude.

Which provider should I pick?

If you just want it to work for free, pick Groq. If you already use Google AI Studio, Gemini is just as good. The two paid options are for users who want premium quality.

How to create your key

Groq Free

  1. Go to console.groq.com and sign in (Google or GitHub).
  2. Open the left sidebar → API Keys.
  3. Click Create API Key, give it a name (e.g. "PromptVault"), and copy the key — it starts with gsk_….
  4. Paste it into PromptVault → Settings → AI → API Key, and pick a Groq model (Llama 3.3 70B is a good default).

Gemini (Google AI Studio) Free

  1. Go to aistudio.google.com/apikey and sign in with your Google account.
  2. Click Create API key → Create API key in new project.
  3. Copy the key (it starts with AIza…).
  4. Paste it into PromptVault → Settings → AI → API Key, and pick a Gemini model (Flash is the cheapest/fastest).

OpenAI Paid · $5 min

  1. Go to platform.openai.com/api-keys and sign in.
  2. Add at least $5 of credit from Billing (required before you can use the API).
  3. Click Create new secret key, copy it (starts with sk-…) — you won't be able to see it again.
  4. Paste it into PromptVault → Settings → AI → API Key, and pick an OpenAI model (GPT-4o-mini is the cheapest option).

Anthropic (Claude) Paid · $5 min

  1. Go to console.anthropic.com and sign up.
  2. Add at least $5 of credit from Plans & Billing.
  3. Open API Keys → Create Key and copy the value (starts with sk-ant-…).
  4. Paste it into PromptVault → Settings → AI → API Key, and pick a Claude model (Haiku is cheapest, Sonnet is best quality).

Where to paste it in the extension

  1. Open the PromptVault side panel on any AI chat site.
  2. Click the Settings gear at the top right.
  3. Open the AI section.
  4. Pick your provider, paste the key, choose a model, and hit Save.

From there, every prompt you select inside the panel can be improved with one click — no monthly cap, you're using your own quota.

Privacy note — your API key is stored locally with the Chrome storage.local API. It's never sent to PromptVault servers; requests go directly from your browser to the provider you chose.
